# 📁 在线文件压缩工具

一个完全在浏览器中运行的在线文件压缩工具，支持多种格式，保护用户隐私。

## 🌐 在线地址

[https://josenxie51-bit.github.io/file-compressor/](https://josenxie51-bit.github.io/file-compressor/)

## ✨ 功能特性

- 🔒 **隐私保护** - 所有文件在浏览器中处理，不会上传到服务器
- 🚀 **快速压缩** - 支持批量处理，即时压缩
- 📁 **多格式支持** - PDF、Word、Excel、图片、文本等
- ⚙️ **自定义设置** - 可调节压缩级别和目标大小
- 📱 **响应式设计** - 适配电脑、手机、平板
- 💯 **完全免费** - 无需注册，无使用限制

## 🛠️ 使用方法

1. **上传文件**
   - 拖拽文件到上传区域
   - 或点击"选择文件"按钮

2. **设置选项**
   - 选择压缩级别（普通/高质量/极限）
   - 设置目标大小限制（可选）
   - 选择输出格式

3. **开始压缩**
   - 点击"开始压缩"按钮
   - 等待处理完成

4. **下载文件**
   - 点击下载链接
   - 保存压缩后的文件
